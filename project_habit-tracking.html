Habit tracking app with Python and MongoDB

Overview

This Habit Tracker is a web application designed to help users build and maintain positive habits through effective tracking and visualization. Built with Python and Flask, this project provides an intuitive interface for users to monitor their daily habits and make meaningful progress toward their personal goals.

The application features a user-friendly design, allowing users to easily create, set daily goals. With MongoDB as the data management solution, we ensure efficient storage and retrieval of user information, providing a scalable framework for habit tracking.

Technologies used

  • Python
  • HTML
  • CSS
