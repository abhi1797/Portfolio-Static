Weather Tracker Golang

Overview

Developed a weather tracking application using Go (Golang) that integrates with the OpenWeatherMap API to provide real-time weather updates. The application features:

  • API Integration: Utilized the OpenWeatherMap API to fetch current weather metrics based on user-defined locations, handling JSON responses for seamless data extraction.
  • Configuration Management: Implemented a secure mechanism for API key management by storing sensitive information in a .apiConfig file formatted in JSON, ensuring secure access during runtime.
